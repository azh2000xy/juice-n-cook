# 瑞士排骨的做法

瑞士排骨是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 净排骨400克
- 马铃薯300克
- 茄汁35克
- 精盐
- 白糖各
- 味精
- 糖醋汁0
- 威化15克
- 绍酒
- 湿淀粉
- 干淀粉各0
- 蒜茸1
- 高汤
- 花生油1000克

## 操作

1. 将马铃薯去皮，切成菱形，用七成热油炸至金黄色捞起待用。 (
2. 将炒锅内油烧成五成热，将威化炸膨化后捞起放在盘里。 (
3. 将排骨斩成段，沾上一层薄干淀粉，下入六成热的花生油锅内炸至金黄色捞出，沥去油。 (
4. 炒锅放油
5. 5汤匙，烧四成热，放入蒜茸、排骨，烹绍酒，爆透后加入高汤，调入茄汁，精盐、味精、白糖、糖醋汁，加盖ju至九成熟时，揭盖再加入炸过的马铃薯略ju至熟，加熟花生油
6. 5汤匙推匀装盘，威化伴边即成。
