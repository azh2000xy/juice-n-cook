# 炸鸡椒的做法

炸鸡椒是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 主料
- 青虾肉200克
- 水发海米10克
- 猪肥膘肉50克
- 鸡蛋150克
- 炸粉丝50克
- 细面包渣
- 生菜叶各100克
- 调料
- 香油5克
- 料酒
- 面粉各10克
- 姜末各15克盐4克

## 操作

1. 剔净鸡胸脯翅骨上的肉，剔除筋皮，在鸡胸脯肉上均匀在用刀尖扦些眼再砸松，然后用盐稍腌一下。鸡蛋打入碗中控出蛋清25克备用，其余作筷子拨散。生菜洗净、消毒待用。 (
2. 把青虾肉和肥膘肉一起剁成茸，加入鸡蛋清，搅入25克清水，再加葱、姜末、料酒、盐、海米末和香油搅成馅，分成12分。 (
3. 把鸡胸脯肉顺切一刀分成两片，但带骨一头不要切断，在每片上放一分馅，卷成纺锤形成为“鸡椒”。 (
4. 将鸡椒蘸上一层面粉，而后再裹上一层鸡蛋液，粘上一层面包渣待用。 (
5. 提起鸡翅骨逐个放入烧至5成热的植物油中炸，使翅骨竖立与鸡肉几乎成9°角，炸5分钟左右油温逐渐升高，把鸡椒炸呈金黄色时捞出沥油。 (
6. 将炸粉丝放在盘内一端嵌上鸡椒，插上萝卜花，将生菜叶围在四周即成。
