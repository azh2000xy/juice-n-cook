# 油爆肚仁的做法

油爆肚仁是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 主料羊肚仁200克
- 青蒜2克
- 蒜片2克
- 调料熟大油600克
- 花椒油16克
- 鸡汤12克
- 湿淀粉8克
- 料酒6克
- 醋2克
- 盐2克
- 味精2克
- 葱2克
- 姜汁5克

## 操作

1. 把羊肚仁去掉油脂，洗干净，切成4×2厘米的肚条。把葱和青蒜都切成
2. 6厘米长的小段，一同放入碗中，加入蒜片、湿淀粉、料酒、醋、盐、姜汁、味精和鸡汤等，调成芡汁。 (
3. 用开水把肚条烫一下。 (
4. 炒勺里倒入大油，放在旺火上烧到冒烟时，放入肚条，爆2秒种，倒在漏勺内沥去油。 (
5. 炒勺内倒入花椒油烧热，放放爆好的肚条，翻炒几下，倒入芡汁后，再翻炒3～4秒钟即成。
