# 花束鸡的做法

花束鸡是一道家常美味。

预估烹饪难度：★★★☆☆

## 必备原料和工具

- 主料 鸡胸脯肉400克
- 方火腿80克
- 大青椒
- 水发大冬菇
- 鸡蛋清120克
- 调料 生菜油800克
- 料酒25克
- 盐5克
- 味精5克
- 湿淀粉10克
- 鸡汤40克

## 操作

1. 剔去鸡胸脯肉筋皮，切长约5厘米的细丝，分成24分(每分约6根)，均用料酒、盐、味精吃好味。将火腿、青椒、冬菇均分别切细丝，各分成24分。 (
2. 用湿粉、蛋清调成糊，取1分鸡丝，抹上蛋糊后，再将火腿丝、冬菇丝、青椒丝各1分并拢，横放在鸡丝顶头。将鸡丝卷起，一端露出三丝，一端是鸡丝卷成的圆形，成为“花束”。其余均照此法逐一做好。 (
3. 炒勺上旺火，将生菜油烧至4～5成热时将花束下勺温熟后倒出。(
4. 原勺放入味精、糖、盐、料酒、鸡汤，烧开后用湿淀粉勾芡，把拉熟的花束鸡倒入勺内翻几下，淋少许油即可。
