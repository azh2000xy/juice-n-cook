# 红烩肉杂拌的做法

红烩肉杂拌是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 熟杂拌肉
- 火腿
- 猪肉)600克
- 葱头250克
- 胡萝卜100克
- 芹菜100克
- 面粉15克
- 生菜油250克
- 大蒜末15克
- 番茄酱100克
- 砂糖15克
- 辣酱油10克
- 鸡清场250毫升
- 于辣椒
- 香叶
- 精盐
- 白酷各适量

## 操作

1. 1.将熟杂拌肉切成片
2. 再把平底锅放入生菜油，烧八成热，将杂拌肉倒入稍炒一下，链出生菜油
3. 2．再将葱头、胡萝卜去皮，洗净，切成片
4. 把芹菜去筋，洗净，切成丝
5. 3．平底锅放少许生菜油，烧热，下入葱头、胡萝卜、芹菜炒到八成熟，再倒入面粉炒熟，然后，依次加入番茄酱、白醋、干辣椒、鸡清汤、辣酱油、精盐、砂糖、大蒜末、香叶，用手勺搅匀，再将炒好的倒入，微开，出锅，晾凉入盘，即成
