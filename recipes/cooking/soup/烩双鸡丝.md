# 烩双鸡丝的做法

烩双鸡丝是一道家常美味。

预估烹饪难度：★★★☆☆

## 必备原料和工具

- 主料生鸡脯80克
- 熏鸡丝60克
- 鲜碗豆30克
- 鸡蛋清30克
- 调料大油600克
- 鸡汤160克
- 湿淀粉16克
- 葱油16克
- 料酒10克
- 酱油5克
- 盐2克
- 味精2克

## 操作

1. 净生鸡脯洗干净，切成丝丝，放入碗内，先加入鸡蛋清，再加入调稀的湿淀粉(8克)，搅拌均匀。把熏鸡丝也切成丝待用。 (
2. 炒勺内倒入大油，在旺火上烧4～5成热，倒入 生鸡丝滑开，捞出沥油。 (
3. 用开水把鲜豌豆焯一下，捞出沥水。 (
4. 炒勺放旺火上、倒入鸡汤，加入熏鸡丝、料酒盐、酱油、味精，开锅后调好色和味，用湿淀粉勾稀芡，再把滑好的生鸡丝、豌豆下入，搅拌均匀，滴入葱油即成。
