# 连锅汤的做法

连锅汤是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 猪后腿肉200克
- 白萝卜75克
- 大白菜均可)
- 辣椒油
- 豆瓣酱
- 葱各13克
- 胡椒粉
- 花椒粉少许
- 料酒13克
- 酱油50克
- 味精3克
- 花椒5粒
- 姜8克

## 操作

1. 肉用开水煮透后捞出，切成大薄片，萝卜切
2. 2厘米厚、3厘米宽4厘米长的片、姜柏破，整葱。 (
3. 用酱油、辣椒油、花椒粉、豆瓣酱和汤对成汁。 (
4. 将萝卜下入原汤加花椒和葱，大火煮烂后下入肉片，稍煮后下入料酒、味精、胡椒粉、盐即可。食时蘸汁。
