# 甲鱼枸地汤的做法

甲鱼枸地汤是一道家常美味。

预估烹饪难度：★★☆☆☆

## 必备原料和工具

- 甲鱼
- 枸杞子35克
- 熟地黄15克
- 绍酒10克
- 生姜12克
- 味精1克
- 精盐3克

## 操作

1. 制法：将甲鱼放入开水内烫死，刷洗几遍，斩头、爪，揭去甲壳，挖出内脏，放入清水盆内洗净，切成大块，待用
2. 将生姜去外皮，清火洗净，切成片，待用
3. 把枸杞子、熟地黄用清水冲净，待用
4. 把沙锅刷洗净，加清水适量，置于旺火上烧沸，将甲鱼肉、生姜片、枸杞子、熟地黄、绍酒齐放入锅内，锅加盖，用文火慢煮2小时，点入精盐、味精调味，即可上桌供食用
