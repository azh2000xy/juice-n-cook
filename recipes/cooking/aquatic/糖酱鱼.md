# 糖酱鱼的做法

糖酱鱼是一道家常美味。

预估烹饪难度：★★☆☆☆

## 必备原料和工具

- 主料 鲜鲤鱼
- 白糖200克
- 辅料 酱油0
- 姜片
- 醋
- 植物油800克

## 操作

1. 1．将鱼洗净，去鳞、鳃，膛去内脏，切成大块
2. 2．锅内放植物油，烧至八成热时，把鱼放入锅内炸
3. 两面都炸黄（不必太焦）时，捞出，控净油
4. 3．把锅内油倒出，放入炸好的鱼和酱油、白糖、姜、醋，在旺火上烧开后，用小火慢炖，炖至汤干即成
