# 虫草杜仲炖海虾的做法

虫草杜仲炖海虾是一道家常美味。

预估烹饪难度：★★☆☆☆

## 必备原料和工具

- 新鲜海虾1斤
- 冬虫夏草3钱
- 生杜仲3钱
- 酒1小匙
- 盐1小匙

## 操作

1. 海虾留壳剪去须、脚，挑去泥肠，洗净沥乾。 (
2. 冬虫夏草、生杜仲以清水洗净杂质。 (
3. 取5碗水加虫草、生杜仲先熬汤。 (
4. 约熬40分钟後，再放进处理好之虾，并加酒调味去腥，虾熟後，加盐即成。
