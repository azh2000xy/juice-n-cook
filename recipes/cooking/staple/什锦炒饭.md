# 什锦炒饭的做法

什锦炒饭是一道家常美味。

预估烹饪难度：★★★★☆

## 必备原料和工具

- 大米饭250克
- 瘦猪肉
- 熟火腿
- 水发海鲜各20克
- 水发香菇5克
- 罐头冬笋和罐头青豆各10克
- 鸡蛋
- 熟鸡脯肉15克
- 花生油35克
- 另备少许酱油
- 料酒
- 精盐
- 味精和葱白

## 操作

1. 制法：猪肉剁成末
2. 火腿、海参、香茹、冬笋、鸡脯肉均切成小丁
3. 葱白切葱花
4. 鸡蛋在碗内搅匀
5. 炒锅放入花生油15克烧热，倒入蛋液，炒熟搅碎倒出，锅内放20克花生油烧热，下肉末和葱花煸炒一下，加酱油和料酒煸炒几下，放入海参、冬笋、香菇、青豆、熟火腿和鸡脯肉，炒均匀，再倒入米饭和炒好的鸡蛋，加精盐、味精炒透，出锅盛入盘内即成
