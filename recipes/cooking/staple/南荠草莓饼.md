# 南荠草莓饼的做法

南荠草莓饼是一道家常美味。

预估烹饪难度：★★☆☆☆

## 必备原料和工具

- 荸荠500克
- 草莓酱100克
- 面包渣100克
- 白糖45克
- 水淀粉25克
- 干淀粉10克
- 鸡蛋清50克
- 油适量

## 操作

1. 将荸荠去皮后用刀背拍碎，块大的再剁开，用豆包布包好拧干水分，加入干淀粉搅拌均匀，和成面团。 (
2. 将面团分块包上草莓成球状，逐个粘上鸡蛋清、面包渣拍实。 (
3. 起锅放油投入荸荠饼坯，炸至金黄色捞出码盘按扁。 (
4. 净锅加入水、白糖烧开，用水淀粉勾玻璃芡，淋明油，浇在饼上即可。
